Sistema operativo
Es el software que administra los recursos del computador y permite ejecutar programas. También facilita la comunicación entre el hardware y las aplicaciones.

Node.js
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. 
Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js
Es el archivo que contiene el código del servidor. Define cómo responder a las solicitudes que realizan los
usuarios.

Puerto
Es un punto de comunicación que utiliza el servidor para recibir conexiones de los clientes. Cada servicio funciona en un puerto específico.

Logs
Son los mensajes que muestra el servidor en la terminal durante su ejecución. Sirven para verificar el funcionamiento y detectar errores.
